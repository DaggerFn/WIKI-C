Principais Bibliotecas do C

1. stdio.h (Standard Input/Output)

Fornece funções para manipulação de entrada e saída padrão, incluindo leitura e escrita em arquivos.

Funções Principais:

  • Printf(); - A função printf exibe um ou mais dados na tela. Para tanto ele deve receber pelo menos dois parâmetros, separados por vírgula: um string de formato que define, através de caracteres especiais, os tipos dos dados a serem impressos e suas posições na linha de impressão; um dado a ser impresso.

  • Scanf(); - A principal função de entrada de dados da Linguagem C é scanf. Para usá-la você precisa definir a string de formato e as variáveis onde os dados lidos serão armazenados. permite que se leia um int, um float e um int, nesta ordem.

  • Fopen(); - Abre um arquivo texto para leitura. O arquivo deve existir antes de ser aberto.

  • Fclose(); - determina o arquivo a ser fechado. A função retorna zero no caso de sucesso. Fechar um arquivo faz com que qualquer caracter que tenha permanecido no "buffer" associado ao fluxo de saída seja gravado.

  • Getchar(); - obtém sua entrada do teclado. Portanto, quando o programa acima é executado, o programa espera que o usuário digite alguma coisa. Cada caractere digitado é mostrado no monitor. - O usuário pode digitar diversos caracteres na mesma linha, inclusive backspace para corrigir caracteres já digitados.

  • Putchar(); - coloca o caractere c na tela. Este caractere é colocado na posição atual do cursor. Mais uma vez os tipos são inteiros, mas você não precisa se preocupar com este fato.

  • Fgets(); - retorna um ponteiro para o buffer string se bem-sucedido. Um valor de retorno NULL indica um erro ou uma condição de fim de arquivo.

  • Fputs(): - (O nome é uma abreviatura de file put string) da biblioteca stdio grava uma string num arquivo. Por exemplo, fputs (str, arq); grava a string str (exceto seu byte nulo final) no arquivo arq.

2. stdlib.h (Standard Library)

Oferece funções para alocação de memória, controle de processos, conversão de tipos, geração de números aleatórios e mais.

Funções Principais:

  • Malloc(); - (O nome é uma abreviatura de memory allocation) aloca espaço para um bloco de bytes consecutivos na memória RAM (= random access memory) do computador e devolve o endereço desse bloco. O número de bytes é especificado no argumento da função.

  • Free(); - Desaloca um bloco de memória ( memblock ) que foi alocado anteriormente por uma chamada a calloc, malloc ou realloc.

  • Atoi(); - Retorna um valor int produzido interpretando os caracteres de entrada como um número. O valor de retorno será 0 se a função não puder converter a entrada em um valor desse tipo. O valor de retorno é indefinido no caso de um estouro.

  • Atof(); - Essa função permite converter a string pN em double, onde pN é um ponteiro para uma constante de caractere. Essa função retorna um valor double e, caso o valor convertido não puder ser representado, o comportamento da função será indefinido.

  • Rand(); - Produz um valor aleatório na faixa entre 0 e a constante RAND_MAX. O valor desta constante encontra-se definida no arquivo stdlib. h. Muitas vezes necessitamos gerar valores dentro de determinada faixa.

  • Srand(); - Define o ponto de partida para gerar uma série de inteiros pseudoaleatórios no thread atual. Para reinicializar o gerador para criar a mesma sequência de resultados, chame a função srand e use o mesmo argumento seed novamente.

  • Exit(); - Define um ponto de interrupção no ponto de saída especificado no bloco especificado.

  • System(); - Chama o console do sistema operacional (se já não estiver rodando) e executa um comando neste aplicativo. No caso está chamando o comando pause do sistema operacional que fará o console ficar esperando por uma tecla e consequentemente sua aplicação esperará por isso também.

3. string.h

Contém funções para manipulação de strings e arrays de caracteres.

Funções Principais:

  • Strlen(); - É uma função para encontrar o comprimento de uma string. Conta o total de caracteres apresentados em uma string, eliminando o caractere nulo. O número total de caracteres na string inclui alfabetos, caracteres especiais e numbers, com espaços em branco.

  • Strcpy(); - É uma função de biblioteca disponível na biblioteca de strings em C e é usada para copiar o conteúdo de uma string para outra. A função strcpy() não cria uma string (matriz de caracteres), ela apenas copia uma string (matriz de caracteres).

  • Strncpy(); - Copia a contagem de caracteres de string2 para string1 . Se a contagem for menor ou igual ao comprimento da string2, um caractere nulo (\0) não será anexado à string copiada.

  • Strcat(); - A primeira letra da string fonte substitui o caractere terminal da string de destino, e após toda a string fonte ser concatenada, o caractere terminal é inserido no final da string de destino.

  • strncat(); - A função da biblioteca C char *strncat(char *dest, const char *src, size_t n) anexa a string apontada por src ao final da string apontada por dest até n caracteres.

  • strcmp(); - Na biblioteca padrão, a função strcmp() faz distinção entre letras maiúsculas e minúsculas. Se você não quer que a função faça esta distinção, você pode modificar o seu string para ter apenas letras minúsculas (ou maiúsculas) antes de passá-lo como argumento para a função strcmp().

  • strncmp(); - Faz distinção entre letras maiúsculas e minúsculas. Se você não quer que a função faça esta distinção, você pode modificar o seu string para ter apenas letras minúsculas (ou maiúsculas) antes de passá-lo como argumento para a função strcmp().

  • strstr(); - Retorna um ponteiro para a primeira ocorrência de strSearch em str . A pesquisa não inclui o encerramento de caracteres nulos. wcsstr é a versão de caractere largo de strstr e _mbsstr é a versão de caracteres multibyte. Os argumentos e o valor de retorno de são cadeias de wcsstr caracteres largos.

  • memset(); - Preenche uma quantidade de uma determinada área da memória com um dado valor. em outras palavras, inicializa algum objeto (variável, estrutura, etc).

  • memcpy(); - É usada para copiar um bloco de memória de um local para outro . Ao contrário de outras funções de cópia, a função memcpy copia o número especificado de bytes de um local de memória para outro local de memória, independentemente do tipo de dados armazenados. É declarado em arquivo de cabeçalho.

  • memcmp(); - É usada para comparar os primeiros n bytes de duas áreas de memória. Ela compara os valores desses bytes e retorna um inteiro que indica a relação entre os buffers.

4. math.h

Proporciona funções matemáticas comuns, incluindo operações aritméticas, trigonométricas e exponenciais.

Funções Principais:

  • sin(); - Retorna um valor numérico entre -1 e 1, que representa o seno de um angulo dado em radianos. Como sin() é um método estático de Math , você sempre o usa como Math. sin() , ao invés de um método de um objeto Math que você criou ( Math não é um construtor).

  • cos(); - A função cos() calcula o cosseno de x. O valor x é expresso em radianos. Se x for muito grande, poderá ocorrer uma perda parcial de significância no resultado.

  • tan(); - Retorna a tangente de um número (ângulo em radianos).

  • asin(); - Função recebe um único argumento (1 ≥ x ≥ -1) e retorna o arco seno em radianos.

  • acos(); - A função da biblioteca C double acos(double x) retorna o arco cosseno de x em radianos.

  • atan(); - A função ATAN retorna o arco tangente do argumento como um ângulo expresso em radianos. As funções ATAN e TAN são operações inversa. Uma expressão que retorna um valor de qualquer tipo de dado numérico embutido (exceto para o DECFLOAT), cadeia de caracteres ou tipo de dados de cadeia gráfica.

  • exp(); - A função exp retorna o valor do expoente (ex).

  • pow(); - Retorna a base elevada ao expoente power, ou seja, baseexpoente.

  • sqrt(); - Retorna a raiz quadrada de um número.

  • floor(); - retorna o menor número inteiro dentre o número "x". Já o Math. round ele retorna o valor de um número arredondado para o inteiro mais proximo.

  • fabs(); - Calcula o valor absoluto do argumento de ponto flutuante x.

5. time.h

Inclui funções para manipulação e obtenção de tempo e data.

Funções Principais:

  • time(); - Retorna o número de segundos decorridos desde a meia-noite (00:00:00), 1º de janeiro de 1970, UTC (Tempo Universal Coordenado), de acordo com o relógio do sistema. O valor retornado é armazenado no local fornecido por destTime .

  • difftime(); - F'unciona em intervalos de tempo numéricos ou codificados por caracteres ; neste último caso, as unidades devem ser especificadas e o formato não tem efeito. A aritmética limitada está disponível em objetos "difftime": eles podem ser adicionados ou subtraídos e multiplicados ou divididos por um vetor numérico.

  • clock(); - Informa quantas horas do relógio se passaram desde a inicialização do CRT durante o início do processo. Essa função não está estritamente em conformidade com a ISO C, que especifica o tempo líquido da CPU como o valor de retorno. Para obter tempos de CPU, use a função Win32 GetProcessTimes.
  • mktime(); - As funções mktime , _mktime32 e _mktime64 convertem a estrutura de horário fornecida (possivelmente incompleta) apontada por timeptr em uma estrutura completamente definida com valores normalizados e a converte em um valor temporal de calendário time_t.

  • strftime(); - Coloca bytes na matriz apontada por s como controlados pela cadeia apontada por format. A cadeia de formatação consiste em zero ou mais especificações de conversão e caracteres comuns.

  • asctime(); - Converte uma hora armazenada como uma estrutura para uma cadeia de caracteres. O timeptr valor é normalmente obtido de uma chamada para ou localtime , que retornam um ponteiro para gmtime uma tm estrutura, definida em TIME.
  • ctime(); - Retorna o número de segundos decorridos desde a meia-noite (00:00:00), 1º de janeiro de 1970, UTC (Tempo Universal Coordenado), de acordo com o relógio do sistema. O valor retornado é armazenado no local fornecido por destTime .
